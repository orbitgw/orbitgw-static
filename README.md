# ORBITGW's Resources Hosting service

A CDN service for hosting commonly used frontend static resources.

## Features

- **High-speed Access**: Cloudflare CDN acceleration to boost your speed
- **Version Management**: All resources are organized by version numbers for easy rollback

## Examples

Using Bootstrap 5.3.8:

```html
<link rel="stylesheet" href="https://cdn.orbitgw.com/static/bootstrap/5.3.8/css/bootstrap.min.css">
<script src="https://cdn.orbitgw.com/static/bootstrap/5.3.8/js/bootstrap.bundle.min.js"></script>
```
